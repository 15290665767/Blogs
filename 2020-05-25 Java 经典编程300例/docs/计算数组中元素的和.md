## 计算数组中元素的和
```
public class Example10 {
  public static void main(String[] args) {
    int [] arr = { 85, 65, 86, 88, 38, 69, 100 };
    Integer totel = 0;
    Integer index = 0;
    while (arr.length > index) {
      totel += arr[index];
      index++;
    }
    System.out.println(totel);
  }
}
```
